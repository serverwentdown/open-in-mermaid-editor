<p align="center">
  Open embedded diagrams in a Mermaid editor
</p>

---

## Usage

- Right-click on a diagram
- Select "Edit Mermaid"
- Choose the online editor or copy the code to your clipboard

## Supported

- Embedded images, links and frames to
  - mermaid.ink
  - kroki.io
